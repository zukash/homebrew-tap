# Homebrew Tap for zukash's Tools

This tap provides Homebrew formulas for various command-line tools.

## Available Formulas

### [notif](https://github.com/zukash/notif)

Minimal macOS Notification Center controller for keyboard shortcuts.

**Install:**
```bash
brew install zukash/tap/notif
```

**Commands:**
- `notif expand` - Expand notification stack
- `notif collapse` - Collapse notification stack
- `notif click` - Click first notification
- `notif close` - Close first notification

**Learn more:** [github.com/zukash/notif](https://github.com/zukash/notif)

---

## Usage

Install any formula directly with:

```bash
brew install zukash/tap/<formula-name>
```

Or tap first, then install:

```bash
brew tap zukash/tap
brew install <formula-name>
```

To update formulas:

```bash
brew update
brew upgrade <formula-name>
```
